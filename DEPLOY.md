# Deploy Guide (Free)

## Option A: GitHub Pages
1. Create a GitHub repo (public).
2. Upload these files:
   - index.html
   - style.css
   - script.js
   - terms.html
   - privacy.html
   - LICENSE.txt
   - favicon.svg
   - social-card.png
   - apple-touch-icon.png
3. Repo → Settings → Pages:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
4. Your site will appear at:
   https://<username>.github.io/<repo>/

## Option B: Netlify
1. Drag-and-drop the folder into Netlify “Deploy manually”.
2. Netlify gives you a live URL instantly.
3. Optional: connect a custom domain.
